The **Hard Stack** is a typed, end-to-end TypeScript monorepo on Bun + Turbo.

<CardGrid stagger>
  <Card title="Type-safe API" icon="setting">
    tRPC routers double as an OpenAPI surface via `trpc-to-openapi`, rendered
    with Scalar.
  </Card>
  <Card title="Auth & DB" icon="approve-check">
    Better Auth on top of Drizzle + Postgres, with email/password and
    verification emails via Resend.
  </Card>
  <Card title="Modern frontend" icon="laptop">
    React 19 + Vite, TanStack Router & Query, and Tailwind v4.
  </Card>
  <Card title="One toolchain" icon="rocket">
    Bun workspaces, Turbo pipelines, Biome, and Lefthook hooks.
  </Card>
</CardGrid>
